Mission-Critical Projects

ERC 2025 // Autonomous Mars Rover

Lead Systems Engineer for a 6-DOF robotic planetary exploration platform.

01 // The Problem

Planetary rovers require deterministic navigation in unstructured, high-latency environments. Existing commercial solutions lacked the required level of tight integration between the ROS2 navigation stack and custom-built hardware joints, leading to cumulative error in complex manipulation tasks.

02 // Engineering Approach

We implemented a **distributed compute architecture** using a central Linux companion computer for SLAM (RTAB-Map) and multiple ESP32 nodes for real-time sensor fusion and joint control. This decoupled high-level path planning from low-latency motor control, ensuring system stability even during network drops.

UAV System // Precision Autonomy

Development of a GPS-denied autonomous landing system for multi-rotor platforms.

01 // The Problem

Standard GPS-based landing systems exhibit drift exceeding 2 meters, making automated recovery on moving or small platforms impossible. High-fidelity autonomy requires sub-decimeter precision in environments where GPS signal may be intermittent or degraded.

02 // Engineering Approach

We integrated a **vision-based odometry stack** running YOLOv8 on a companion PC (Jetson/RPi) paired with ArUco marker detection. By fusing visual data with RTK-GPS telemetry via a custom MAVLink bridge, we achieved deterministic landing control with a 10cm error margin.
